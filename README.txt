# 概要

鈴木個人のPCツール環境をポータブル化したものです。
別のPCにて即時展開することができます。

メインPC環境の自動同期の仕組みについて気になる方はこちら (技術者向け)
https://www.youtube.com/watch?v=IAKbhnj8J14


# 含まれているアプリケーション

1. AutoHotkey...ショートカットキーを自作できる。鈴木のマイセットが設定されている。
2. MouseGestureL...マウスの動きでPC操作できる。
3. Clibor...コピーした文章の履歴が保存され、いつでも再利用できる。
4. Kazaguru...Altキーを押している間、ウィンドウのどこをドラッグしてもウィンドウが動かせる。


# 使用前の準備

1. Zipファイルを解凍(展開)する。
2. フォルダをデスクトップかどこかに移動して、見失わないようにする。
3. 使いたいアプリケーションのリンクを開く。
   「AutoHotkey.lnk」など。
   すべてのアプリケーションを使いたい場合、「AppSwitcher.bat」を起動する。

# 終了方法

1. 右下のタスクトレイから、それぞれのアプリケーションを右クリック→終了とする。
2. すべて起動している場合は、再度「AppSwitcher.bat」を起動することですべて終了できる。

# AutoHotkeyの使い方

IT中級者向け解説
https://ashiatoblog.com/posts/pc/autohotkey/ahk1/

技術的な解説
https://qiita.com/ryoheiszk/items/092cc5d76838cb5a13f1

詳しくは、「ProgramFiles\AutoHotkey\Plugins」内のスクリプトを参照されたし。
多くの機能を付与しているが、基本的には、まず下記が使えれば便利だと思われる。

無変換+I:↑
無変換+J:←
無変換+K:↓
無変換+L:→
無変換+u: ↑↑↑↑
無変換+m: ←←←←
無変換+,: ↓↓↓↓
無変換+.: →→→→

無変換+h: Home
無変換+;: End

無変換+n: Backspace
無変換+/: Delete

無変換+z: 0
無変換+x: 1
無変換+c: 2
無変換+v: 3
無変換+s: 4
無変換+d: 5
無変換+f: 6
無変換+w: 7
無変換+e: 8
無変換+r: 9
無変換+g: +
無変換+t: -
無変換+a: *
無変換+q: /
無変換+b: =
無変換+Tab: Enter
無変換+半角: Escape

無変換+↑: 「↑」を入力
無変換+←: 「←」を入力
無変換+↓: 「↓」を入力
無変換+→: 「→」を入力
