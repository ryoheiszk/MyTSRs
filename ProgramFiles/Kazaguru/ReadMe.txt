  かざぐるマウス

------------------------------------------------------------------------------


●概要
  マウスの操作で様々な拡張機能を実行したり、ブラウザ等でマウス ジェスチャーを
  利用できるようにする Windows 用ソフトウェアです。


●主な機能
  ・マウス カーソル直下のスクロール
  ・ホイールによるウィンドウの前面化
  ・スクロールバー上で水平スクロール
  ・スクロールの加速
  ・修飾キーによる拡張スクロール
  ・マウス ジェスチャー
  ・ホイールによるウィンドウ操作や音量調節
  ・タスクバー ボタンの並べ替え（XP/Vista）

  この他にも様々な機能があります。


●対応環境
  Windows 8 / 7 / Vista / XP / 2000
